Go Webserver
By Moonstreet Notebook |  Feb 10, 2019  | golang
The last Golang post was a little bit lame so let’s do something more interesting now. Let’s create a webserver. First create a new project structure: mkdir -p ~/go/src/httpserver Then create this file: cat > ~/go/src/httpserver/main.go <<EOF package main import "net/http" func main() { http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) { w.Write([]byte("<h1>Go webserver</h1>")) }) http.ListenAndServe(":8000", nil) } EOF We now execute: cd ~/go/src/httpserver go build main.go ./main Then browse to localhost:8000.
