Windows Defender Realtime Protection Check

4 Jan, 2019 · Read in about 1 min · (41 Words)
cybersecurity

Windows Defender

is tegenwoordig een prima anti-virus programma, mits real-time protection werkt. Check of real-time protection werkt en niet wordt geblocked door een firewall via onderstaand commando:

C:\Program Files\Windows Defender>mpcmdrun -validatemapsconnection

p.s. werkt alleen in een command prompt met admin privileges.
